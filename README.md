# A Setup Guide FOR NEOVIM / TMUX

by Xingyu Zhu  

## A Rapid Installation Mode

Clone this repo:
```sh
git clone https://github.com/MiseryForMe/nvim_configration.git
```


1.Quick installation
```sh
cd nvim-config
bash Nvim-install.sh
```

2.Open nvim, and
```sh
:PlugInstall
```

And it would be all done~
